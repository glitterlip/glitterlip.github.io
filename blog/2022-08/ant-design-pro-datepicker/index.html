ant design pro datepicker
  • Aug 16, 2022
  • One minute read

使用ant design pro的protable时，数据的字段是时间，筛选的时候想按照时间段来筛选

比如订单列表，支付时间字段的column列定义的valueType是dateTime,展示的时候展示成支付时间，但是在列表头筛选的时候按照支付时间段筛选

这个需求可以定义两个字段

{
  title: '支付时间',
      dataIndex: 'paid_at',
    ellipsis: true,
    valueType: 'dateTime',
    hideInSearch: true
},
{
  title: '支付时间',
      dataIndex: 'paid_at',
    ellipsis: true,
    valueType: 'dateTimeRange',
    hideInTable: true,
    search: {
  transform: (value: any) => ({startTime: value[0], endTime: value[1]}),
}
},

筛选查询的时候定义为dateTimeRange，然后hideInTable:true在列表中隐藏这个字段

然后定义一个dateTime，在表头筛选栏中隐藏即可
